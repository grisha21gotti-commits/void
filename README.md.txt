Аннализ создания maсcivа в Python, Java, C++
В Python всё просто: sozdаёш список черёз [9, 99, 999] и можёш сразю брать иль добавливать элемет.

В С++ и Ява непривично: масивы фиксированы (int arr [3] или int [] arr), добавит элемент в сердиту нельзя, только в конецерце перепел другие векторы иль АррейЛист.

Стек Работает Одинаково во вих язек: последний добавлённый элемент достэтсья перевышний. В C++ ethoto stack с push & pop, в Java — Stack с push, pop & peek. В Python это приложение и поп, но не такая пропорция синтаксис.

В целоме, идея та же, что и в Питоне: масив для прямого доступа, стэк для работы свежеух вниз. После Python C++ и Java с уважением
